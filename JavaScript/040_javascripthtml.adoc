== Appendix - JavaScript and HTML interaction

[[javascripthtml_dom]]
=== HTML DOM

A HTML webpage is represented via a HTML Document Object Model (DOM). 
The HTML DOM is defined via a W3C standard. 
This HTML DOM defines a standard way to access and manipulate HTML documents. 
You can use JavaScript to modify the HTML DOM.
